Ptakopysk

C++ game prototyping framework based on component system

How-to:

All you will want to do with this library in your project is:

  • use this template to make main.cpp: TemplateMain.cpp
  • extend your game prototype by adding components made by yourself, based on these templates: TemplateComponent.h, TemplateComponent.cpp

Examples:

  • Floppy Disk
  • Math isn't heartless

Using 3rd-party libraries:

  • Xenon Core 3 - base code structure
  • SFML 2.1 - graphics & audio
  • Box2D - physics
  • JsonCpp - serialization

About used environment:

  • Compiled with: GCC (tdm-1) 4.7.1
  • IDE: Code::Blocks
