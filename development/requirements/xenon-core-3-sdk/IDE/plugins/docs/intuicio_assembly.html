Intuicio Assembly Documentation
version 3.8
!intuicio
3.8language version used. This parameter is optional and when it's ommited then compiler will use the largest version supported.
Every Intuicio script must start with this header.
!start
Begin of code section. Must end with !exit
!exit
End of code section. Must follow !start
!registers-i
countInteger registers count.
Determines number of integer registers. You can use this command multiple times in program and compiler will use the largest value.
!registers-f
countFloat registers count.
Determines number of float registers. You can use this command multiple times in program and compiler will use the largest value.
!stack
sizeSize of the call stack in bytes.
Determines size of call stack in bytes. You can use this command multiple times in program and compiler will use the largest value.
!data
typeType of data value. Possible types: int, float, bytes.
aliasName of data value.
valueInitial data value. For int: 10; float: 3.14; bytes: "string", 0.
Maps given alias to data section and fills it's memory with initial value.
!jump
aliasName of jump space.
Maps given alias to data section and fills it's memory with initial value.
!external
aliasName of external pointer.
Maps given alias to external pointer bound to Virtual Machine before running program.
!struct-def
aliasName of defined structure.
Begins structure definition.
!struct-end
End of structure definition.
!field
typeType of defined field. You can use base types: int, float, byte; or another already defined structure type.
aliasName of defined field.
countField size (array of fields if count > 1).
Defines field inside current structure definition.
!namespace
aliasName of pushed name space.
Push name space. All aliases defined inside curent name space are like: namespace/aliasname.
!namespace-end
Pop name space.
!namespace-using
aliasName of used name space.
Use name space. All aliases defined inside current name space will be searched also in given name space.
